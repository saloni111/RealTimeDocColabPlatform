# Demo Video Setup

## Adding Your Demo Video

1. **Copy your video file** from Desktop to this project:
   ```bash
   # From your Desktop, copy the screen recording
   cp "Screen Recording 2025-09-02 at 9.40.11 PM.mov" ~/Desktop/Projects/collaborative-doc-platform-main/dochub-demo.mov
   ```

2. **Add the video to git:**
   ```bash
   git add dochub-demo.mov
   git commit -m "add-demo-video"
   git push origin main
   ```

3. **The README will automatically reference** the video file
